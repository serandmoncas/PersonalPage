## Por qué esta página

Construí esta página como un espacio propio en internet — un lugar donde escribir sobre lo que aprendo,
mostrar mis proyectos, y experimentar con nuevas tecnologías sin pedirle permiso a nadie.

## El stack

Este sitio está construido con **Next.js 15**, **TypeScript**, y **Tailwind CSS 4**.
El contenido del blog vive como archivos `.mdx` en el repo — sin CMS externo, sin base de datos,
sin nada que pueda romperse a las 2am.

```bash
# Clonar y correr localmente en 3 comandos
git clone https://github.com/serandmoncas/PersonalPage
npm install && npm run dev
```

## Qué esperar aquí

Voy a escribir sobre desarrollo de software, inteligencia artificial, y las cosas que aprendo
construyendo proyectos reales. Sin promesas de publicación semanal — solo cuando tenga algo
interesante que decir.

Si quieres construir tu propia página con este mismo template, puedes aprender cómo en el curso
que está disponible en [sergiomonsalve.com](https://sergiomonsalve.com).
